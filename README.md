# Парсер strings.xml в JSON

Данный скрипт поможет вам извлечь диалоги из файла `strings.xml` в файл формата `.json`

Так-же скрипт подсчитывает количество реплик, слов и символов у конкретного персонажа.

---

### Использование:

1. Открыть командную строку, перейти в папку, где находится файл и запустить:
```sh
python ./DialogParse.py
```

2. Выбрать желаемый strings.xml файл.
3. Увидеть результат.

> [!WARNING]
> В случае если вы используете Linux, то нужно установить tkinter через:
> ```sh
> sudo apt install python3-tk
> ```
