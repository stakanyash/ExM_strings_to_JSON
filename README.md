# Парсер strings.xml в JSON

Данный скрипт поможет вам извлечь диалоги из файла `strings.xml` в файл формата `.json`

Так-же скрипт подсчитывает количество реплик, слов и символов у конкретного персонажа.

---

### Использование:

1. Установить Python версии минимум [3.12.6](https://www.python.org/downloads/release/python-3126/), если он ещё не установлен.

2. Открыть командную строку, перейти в папку, где находится файл и запустить:
```sh
python ./DialogParse.py
```

3. Выбрать желаемый strings.xml файл.
4. Указать нужна ли сортировка по персонажам. 
5. Увидеть результат.

Результат сохраняется **в папку, где расположен strings.xml** в формате `[имя_файла]_dialogues.json`.

> [!WARNING]
> В случае если вы используете Linux, то нужно установить tkinter через:
> ```sh
> sudo apt install python3-tk
> ```
